# 📊 NLP Exploratory Data Analysis (EDA) on IMDb Movie Reviews

## 📌 Project Overview

This project was completed as part of the **Yuva Internship – Week 1 Task**. The objective was to perform **Exploratory Data Analysis (EDA)** and **Text Preprocessing** on the IMDb Movie Reviews dataset to understand the structure of textual data and prepare it for Natural Language Processing (NLP) applications.

---

## 🎯 Objectives

- Explore the IMDb Movie Reviews dataset.
- Perform Exploratory Data Analysis (EDA).
- Analyze sentiment distribution.
- Analyze review lengths and word frequencies.
- Generate visualizations for better insights.
- Apply text preprocessing techniques.
- Prepare the dataset for future NLP and sentiment analysis tasks.

---

## 📂 Dataset Information

- **Dataset Name:** IMDb Movie Reviews Dataset
- **Source:** Kaggle
- **File Format:** CSV
- **Total Records:** 50,000
- **Features:**
  - **review** – Movie review text
  - **sentiment** – Positive or Negative

The dataset contains an equal number of positive and negative movie reviews, making it suitable for sentiment analysis and text classification.
